# DevBoxSetup

[![Contact me on Codementor](.github/contact-me.svg)](https://www.codementor.io/@jamesdbartlett3?refer=badge)

This repository contains scripts and configuration files to set up a development environment on a new machine quickly. It includes installations for various programming languages, tools, and utilities commonly used in software development.

Note: Many of the scripts and configurations in this repository are specific to the author's personal preferences and workflow. Users should review and modify the scripts as necessary to fit their own needs and environments.

Contributions are welcome! If you have suggestions for additional tools or improvements to the setup process, please feel free to open an issue or submit a pull request.
